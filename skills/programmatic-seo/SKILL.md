# Programmatic SEO

## When to Use This Skill
Use this skill when you want to create SEO-driven pages at scale for 
StorX — using templates and data to generate dozens or hundreds of 
pages that target long-tail search queries.

Always read `skills/storx-marketing-context/SKILL.md` first before 
running this skill.

---

## What is Programmatic SEO and Why it Works for StorX

Programmatic SEO means creating a large number of pages from a single 
template, where each page targets a slightly different keyword or 
audience. Instead of writing one blog post at a time, you build a 
system that produces 50 or 500 pages at once.

This works exceptionally well for StorX because:

- **StorX has many natural page variations** — every competitor, 
  every use case, every industry, and every location is a potential 
  page
- **Decentralized storage is an emerging category** — long-tail 
  keywords have low competition, making it easier to rank
- **Technical buyers search with specific queries** — "S3 compatible 
  storage for SaaS" is more valuable than "cloud storage" even though 
  it gets fewer searches
